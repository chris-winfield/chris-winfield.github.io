Registering for GitHub

On GitHub, follow the on-screen instructions to register. There is both a paid and a gratis subscription to GitHub. For creative writing, the free version is more than adequate.

Once logged in, select "Create repository" on the left side of the screen.

The repository is the highest level of organization for projects on GitHub. Think of it as a big, empty file cabinet where notes, drafts, and other files will be organized. Give it a comfortable name, as it's best left unchanged going forward. Make the repository private. Regardless of the goal of the work, whether it's to be submitted or self-published, making the repository public will allow anyone to see the drafts and notes that are best left for scholars and biographers to study in retrospect. Private will allow for additional users, or collaborators, to join later.

None of the additional options, a README, .gitignore, or license, are necessary for creative writers. Select "Create repository" to finish.

With the repository created, it will list some options to begin utilizing the newly minted space. Git chiefly uses a command line interface, but in the next section, GitHub Desktop and Atom will make for an accessible and fully functional experience friendly to writers.
