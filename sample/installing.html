Installing GitHub Desktop and Atom

GitHub Desktop will be the first step to a simple implementation of Git for creative writers. Download and install GitHub Desktop. From the Welcome screen, select "Sign In to GitHub.com" and return to your web browser. After logging into GitHub, authorize it. After this, GitHub Desktop will open again. On the "Configure Git" screen that appears, leave it as "Use my GitHub account name and email address." Select Finish.

On the right side of the "Let's get started!" screen, select the repository and then select "Clone" below that. Cloning a repository will create a folder on your computer which will contain everything your repository on GitHub contains. GitHub Desktop will fill in the URL of your repository. Below that is the clone location. Because issues can arise from moving this folder, keep it somewhere unobtrusive. After a successful clone, GitHub Desktop can be closed for the time being.

Download and install Atom. After it installs, close Atom and open GitHub Desktop once more. The repository will appear. From the menu bar at the top of the screen, navigate to "Repository" and select "Open In Atom." A restart may be necessary if "Open In Atom" doesn't appear immediately. The repository and its local clone will open in Atom.

Close GitHub Desktop once more, and prepare to make the first commit.

Return to the main page.
